# Note Management Page — RAG-DocTalk

## 页面定位

笔记管理页是用户登录后的首页。功能：查看所有笔记、搜索、新建笔记、切换深色/浅色模式。此处"一个笔记" = 一个文档 + 围绕它的对话 + 它的多模态产物。

---

同 Landing_Page.md 中的色板。新增：

| 角色 | 色值 | 用途 |
|------|------|------|
| 侧栏背景 | `#F6F7F4` rgb(246,247,244) | 左侧导航区，与主内容区微小区分 |

| 深色侧栏 | `#1E201D` rgb(30,32,29) | 深色模式左侧导航区 |

## 逐区域详细设计

### 1. 登录流程

**登录入口在笔记管理页，不在 Landing Page。**

用户在 Landing Page 点击 "开始阅读" → 跳转到笔记管理页 → 检测未登录 → 显示登录弹窗。

登录弹窗：

| 属性 | 值 |
|------|-----|
| 触发 | 页面加载，未检测到 token |
| 外观 | 居中 modal，`max-width: 400px`，背景 `#FFFFFF`，`border-radius: 12px`，无阴影改为 `box-shadow: 0 0 0 1px #E2E5DF`（极细边框代替阴影） |
| 内容 | 标题 "登录 RAG-DocTalk"（思源宋体 22px）+ 邮箱输入框 + 密码输入框 + "登录" 按钮 + 底部 "还没有账号？注册" |
| 输入框 | 思源黑体 15px，`border: 1px solid #E2E5DF`，focus 时 `border-color: #6B9B5E`，`border-radius: 8px`，`padding: 10px 14px` |
| 登录按钮 | 背景 `#6B9B5E`，hover `#597F4E`，白色文字，16px weight 500 |
| 背景遮罩 | `rgba(27,29,26,0.3)` + `backdrop-filter: blur(4px)` |

### 2. 侧栏导航

| 属性 | 值 |
|------|-----|
| 宽度 | `240px` 固定 |
| 背景 | 浅色 `#F6F7F4` / 深色 `#1E201D` |
| 用户区 | 顶部 32px 处：圆形头像(40x40) + 用户名(思源黑体 15px weight 500) + 邮箱(12px `#8B9288`) |
| 导航项 | "所有笔记" "收藏" "设置" — 思源黑体 14px，间距 8px，`padding: 10px 16px`，`border-radius: 8px` |
| 当前选中态 | 背景 `#FFFFFF`（浅色）/ `#2D302B`（深色），文字 `#6B9B5E`，左侧无 stripe border |
| 深色模式开关 | 侧栏底部，按钮式切换：🌙 图标 + "深色模式" 文字（思源黑体 13px `#6B7268`）。点击切换，无弹窗，即时生效。状态持久化到 localStorage |

**关键设计决策：不使用侧边 stripe border 标记选中态。** 用背景色 + 文字色变化区分，更安静。

### 3. 主内容区

| 属性 | 值 |
|------|-----|
| 背景 | 浅色 `#FAFBF8` / 深色 `#1A1C19` |
| 顶部栏 | `padding: 24px 32px`，flex，搜索框在左，新建按钮在右 |
| 搜索框 | 宽度 `320px`，`border-radius: 8px`，`border: 1px solid #E2E5DF`，placeholder "搜索笔记..."，focus 时 `border-color: #6B9B5E`。内嵌搜索图标（`#8B9288`） |
| 新建按钮 | 背景 `#6B9B5E`，hover `#597F4E`，白色 "+ 新建笔记"（思源黑体 14px weight 500），`border-radius: 8px`，`padding: 10px 20px` |

### 4. 笔记卡片网格

| 属性 | 值 |
|------|-----|
| 布局 | `display: grid; grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)); gap: 20px` |
| 卡片 | 背景 `#FFFFFF`（浅色）/ `#242724`（深色），`border-radius: 10px`，无阴影，`border: 1px solid #E2E5DF`（浅色）/ `#2D302B`（深色） |
| hover | `border-color: #6B9B5E`，背景 `#F0F3ED`（浅色）/ `#2D302B`（深色），过渡 200ms ease-out |
| 内边距 | `20px` |

每张卡片内容：

| 元素 | 样式 |
|------|------|
| 标题 | 思源宋体 18px weight 600 `#1B1D1A`，`margin-bottom: 8px` |
| 摘要 | 思源黑体 13px `#6B7268`，单行截断 `text-overflow: ellipsis`，最多一行 |
| 底部元数据行 | flex，思源黑体 12px `#8B9288`。左侧 "12 轮对话"，右侧 "2 小时前"。无图标，纯文字 |
| 更多按钮 | 右上角 `···`，hover 时圆圈背景 `#F0F3ED`，弹出菜单：重命名 / 收藏 / 删除 |

### 5. 空状态

当用户没有任何笔记时：

- 居中显示一个大号 SVG 插图（描边风格，鼠尾草绿灰度，示意文档+对话）
- 下方文字 "还没有笔记"（思源宋体 22px weight 600 `#1B1D1A`）
- 副行 "上传你的第一份文档，开始提问。"（思源黑体 15px `#6B7268`）
- 按钮 "新建笔记"（与顶部栏的新建按钮样式一致）
- 整体 `margin: 120px auto`，文字居中

## 交互细节

| 操作 | 行为 |
|------|------|
| 点击卡片 | 进入该笔记的核心工作界面 (Core Work) |
| 点击 "+ 新建笔记" | 弹出轻量 modal：输入笔记标题(可选) → 点击"创建" → 直接进入 Core Work |
| 搜索 | 输入即过滤，200ms debounce，过滤规则：标题或摘要包含关键词 |
| 深色模式切换 | 瞬间切换（无动画），所有色值立即更换，localStorage 记录偏好 |
| 登录/注册 | modal 内完成，成功后 modal 消失，显示笔记列表 |

## 与 NotebookLM 的差异

| NotebookLM | RAG-DocTalk |
|-------------|-------------|
| 精选笔记本（官方示范案例） | 无官方示范，直接显示用户笔记 |
| 背景图 + 叠加文字的卡片 | 纯文字卡片，靠边框+留白区分 |
| 多种视图模式（网格/列表/紧凑） | 仅网格视图，保持简洁 |
| Google Material Design 蓝灰调 | 鼠尾草绿 + 石板灰书房调 |
| Sans-serif only | 思源宋体标题 + 思源黑体正文 |

## 响应式

| 断点 | 变化 |
|------|------|
| ≥960px | 侧栏 240px + 内容区自适应 |
| 640–960px | 侧栏收起为图标模式(56px)，卡片网格 `minmax(240px, 1fr)` |
| <640px | 侧栏消失，底部 tab bar 代替；卡片单列；搜索框全宽 |
